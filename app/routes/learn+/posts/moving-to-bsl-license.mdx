A few months ago we launched Carbon with an AGPLv3 license. Since that time, we've learned a few things about open-source and manufacturing:

1. Manufacturing companies don't want to open-source their software
2. There's a lot of confusion about our commercial license vs the AGPLv3 license
3. Companies aren't very interested in contributing to the open-source version of Carbon

To eliminate confusion, and build the product that we think our customers want, while protecting ourselves against competitors copying our product, we've decided to move to a BSL license.

Of all the people we've talked to, no one will be affected by this change. If you're using Carbon in the cloud, you'll continue to do so. And if you're self-hosting Carbon to build upon, you can buy a license.

You're still able to use the source code for free, but you'll need to buy a license to use it in production.
